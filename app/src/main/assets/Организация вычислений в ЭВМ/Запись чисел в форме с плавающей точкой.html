Запись чисел в форме с плавающей точкой

Двоичное число может быть записано в виде А=МА ∙ 2ПА

При этом форма записи может быть различной. Для сохранения числа увеличение порядка на 1 должно одновременно приводить к сдвигу содержимого мантиссы на 1 бит вправо.

Запись числа – нормализованное, если его мантисса удовлетворяет условию а-1≤[МА]<1

Например, если МА = 0.00101…,то запись числа – ненормализованная (нарушена нормализация вправо). Для нормализации содержимое мантиссы необходимо сдвинуть в сторону, противоположную нарушению нормализации. В нашем случае – на 2 разряда. При этом содержимое регистра порядка уменьшается на 2. Различают 2 вида сдвигов – простой и модифицированный.

Простой сдвиг выполняется следующим образом:

0аn аn-1… а0 ← аn аn-1… а00 → 00аn аn-1… а0

Модифицированный сдвиг – при котором в сдвигаемы разряд заносится значение совпадающее со значением знакового бита.

Например:

    00 аn аn-1… а0

→00 0   аn … а1

←0 аn аn-1….…2
