Восьмеричная система счисления

Восьмеричная система счисления включает восемь различных цифр для изображения веса разряда 0;1;2;3;4;5;6;7. Таким образом изображение числа в восьмеричной системе:

 

 

Преобразование чисел из восьмеричной системы в десятичную систему осуществляют путём умножения каждого восьмеричного разряда на его позиционный вес:

(165)8 = 1∙82+6∙81+5∙80= 64+48+5 = (117)10

(2,36)8 = 2∙80+3∙8-1+6∙8-2= 2+0,9375+0,094=2,469

Десятичное число преобразуется в восьмеричное путем последовательного деления (для преобразования целой части)или умножения (для преобразования дробной части) на 8:

(37,56)10 → ( )8

 

 

 

 

 

 

(37)10 = (45)8

 

 

 

 

 

(.56)10=(.436)8

Таким образом (37,56)10→(45,436)8

Преобразование записи информации из восьмеричной системы счисления в двоичную выполняется путём конвертирования каждого восьмеричного разряда в двухбитный эквивалент:

(531)8 → ( )2

 

    Тогда (531)8 → (101011001)2
