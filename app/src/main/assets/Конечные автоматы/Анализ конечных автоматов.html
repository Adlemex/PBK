Анализ конечных автоматов

Полное описание поведения автомата заключается в определении последовательности выходных сигналов при возбуждении  его в тактовые моменты времени, некоторой последовательности входных сигналов. Входные и выходные последовательности представляются наборами символов из алфавитов X и Y одинаковой длины e. Для такого описания кроме характеристических функций необходимо определить и задать начальное состояние автомата.

 Удобно определить реакцию автомата на входную последовательность по его графу. Для этого достаточно проследить путь на графе начиная от вершины начального состояния, по направлению дуг, которые отмечены очередными номерами их входной последовательности. Выходная последовательность определяется номерами дуг в порядке их следований по пройденному пути, а последовательность состояний автомата – номерами вершин, через которые проходит этот путь.

 Так для графа для входной последовательности (2, 0, 1, 1, 2, 3) и начального состояния 0 имеем выходную последовательность (0, 1, 0, 0, 1, 1)  и смену состояний автомата (1, 3, 0, 2, 2). При начальной состоянии 2 и той же последовательности получаем (1, 1, 0, 0, 1, 1) и (2,3,0,2,2,3)

 С помощью графа  автомата можно выделить следующие типы его состояний:

 1. Преходящее состояние, из которого можно перейти, по крайней мере, в одно или другое состояние, но после этого уже нельзя возвратиться в его не при каком воздействии (вершина графа не имеет заходящих дуг, но имеет хотя бы одну исходящую дугу)

 2. Тупиковое состояние – в которое можно перейти, по крайней мере, из одного другого состояния, но после этого уже нельзя выйти из него ни при какой воздействии (вершина не имеет исходящих дуг в другие вершины, но имеет хотя бы одну входящую).

 3. Изолированное состояние – из которого нельзя перейти ни в какое другое состояние и в нельзя попасть ни из какого другого состояния (вершина содержит только петлю). Совокупности состояний автомата, соответствующие типам его состояний – подавтоматы.

 Пусть М1, М2, и М3 соответственно переходящий, тупиковый и изолированный подавтоматы автомата M, характеризующиеся множествами состояний S1, S2, и S3. Очевидно, выделение таких подавтоматов соответствует разбиению множества S состояний автомата М на непересекающиеся подмножества S1, S2, и S3.
