Триггер как конечный автомат

В структурном виде триггер представляет собой комбинационную схему с обратной связью:

 Схема может быть описана логическими уравнениями:

 

 1. При отсутствии входных сигналов x1=0 ; x2=0 выходные  определяются неоднозначно уравнениям работы удовлетворяющие решения Y1 = 0; Y2=1 и Y1=1; Y2=0.

 2. Если входные сигналы действуют только на одном из входов x1=0 ; x2=1 или x1=1 ; x2=0, то окончании действия сигнала, т.е. перехода к комбинации x1=0 ; x2=0 выходные величины сохраняют значения, которые имелись ранее, т.е. схема запомнила поданную на ее вход комбинацию входных сигналов.

 3. Если сигналы действуют на оба входа x1=1 ; x2=1, то выходные величины определяются однозначно Y1=0 ; Y2=0, однако по окончанию действия сигналов схема переходит либо к состоянию Y1=0 ; Y2=1, либо к состоянию Y1=1 ; Y2=0 случайным образом.
