Двоичные переменные и булевы функции

Основные понятия логики реализуют действия с двоичными переменными, переключающими (булевыми) функциями. Функция F=f (x1,…,xn) называется булевой, если она и любой из ее аргументов x1,…,xn принимают значения только из множества {0 ; 1}

 Таким образом – область определения булевой функции от n  - переменных - совокупность ее значений для каждого из 2n возможных наборов. Эти значения удобно размещать в таблице истинности в порядке соответствующих двоичных чисел.
